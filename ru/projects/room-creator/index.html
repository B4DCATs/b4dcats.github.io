Room Creator Discord Voice Channel Bot

Discord бот для автоматического создания временных голосовых каналов с интерактивной панелью управления. Python-powered, open source, современные функции.

Добавить на сервер 🤖 Исходный код 💻
room-creator@bot:~$
$ git clone https://github.com/b4dcats/room-creator
Клонирование в 'room-creator'...
$ cd room-creator && pip install -r requirements.txt
Установка зависимостей...
$ python run.py
✅ Room Creator Bot онлайн!

Установка

Начните работу с Room Creator за несколько минут

1

Клонировать репозиторий

git clone https://github.com/b4dcats/room-creator
2

Установить зависимости

pip install -r requirements.txt
3

Настроить окружение

cp .env.example .env
# Отредактируйте .env с вашим Discord токеном
4

Запустить бота

python run.py

Docker установка

Для более простого развертывания и управления

docker-compose up --build -d

Команды

Доступные slash команды и функции

Административные

/setup-lobby <канал>

Настроить лобби канал для создания комнат

/status

Показать статус бота и текущие настройки

Управление комнатами

/room-info

Показать информацию о текущей комнате

/room-transfer <пользователь>

Передать права владения другому пользователю

/room-close

Закрыть текущую комнату (только владелец)
