Twitch Image Resize Инструмент для наград канала

Онлайн-инструмент для изменения размера изображений для наград Twitch канала. Генерация нескольких размеров (28x28, 56x56, 112x112) одним кликом.

Использовать 🖼️ Исходный код 💻
twitch-resize@tool:~$
$ upload image.png
📁 Изображение загружено: 512x512
$ resize --sizes 28,56,112
✅ Генерация размеров: 28x28, 56x56, 112x112
📦 Создание архива...
✅ Архив готов: twitch-rewards.zip
$ _
