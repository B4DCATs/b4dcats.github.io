Installation

Get started with Room Creator in minutes

1

Clone Repository

git clone https://github.com/b4dcats/room-creator
2

Install Dependencies

pip install -r requirements.txt
3

Configure Environment

cp .env.example .env
# Edit .env with your Discord token
4

Run Bot

python run.py

/room-info

/room-transfer <user>
