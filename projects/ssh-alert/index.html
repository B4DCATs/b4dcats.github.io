Quick Installation

1

Clone Repository

git clone https://github.com/B4DCATs/ssh-login-alert cd ssh-login-alert
2

Run Installation

sudo ./install.sh
3

Configure Telegram

Follow the interactive setup to configure your Telegram bot token and chat ID.

What happens during installation?

  • Files are copied to /opt/ssh-alert/ and /etc/ssh-alert/
  • SSH integration is configured through /etc/ssh/sshrc
  • Log rotation configuration is created
  • Interactive Telegram setup is launched
  • Configuration is tested

Configuration

Basic Settings

# Telegram Bot Configuration
TELEGRAM_BOT_TOKEN="your_bot_token_here"
TELEGRAM_CHAT_ID="your_chat_id_here"

# Server Information
SERVER_NAME="server01"
SERVER_DOMAIN="example.com"

# Notification Settings
NOTIFY_INTERACTIVE_SESSIONS=true
NOTIFY_TUNNELS=false
NOTIFY_COMMANDS=false
DISABLE_NOTIFICATION_SOUND_FOR_TUNNELS=true

# Rate Limiting (seconds)
RATE_LIMIT_PER_IP=300
RATE_LIMIT_PER_KEY=60
