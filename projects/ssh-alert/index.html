Configuration

Complete configuration reference

Basic Settings

TELEGRAM_BOT_TOKEN
Your Telegram bot token from @BotFather
TELEGRAM_CHAT_ID
Chat ID for receiving notifications
SERVER_NAME
Server identifier in notifications
SERVER_DOMAIN
Server domain for notifications

Notification Settings

NOTIFY_INTERACTIVE_SESSIONS
Enable notifications for interactive shells
NOTIFY_TUNNELS
Enable notifications for SSH tunnels
NOTIFY_COMMANDS
Enable notifications for command execution
DISABLE_NOTIFICATION_SOUND_FOR_TUNNELS
Disable sound for tunnel notifications

Rate Limiting

RATE_LIMIT_PER_IP
Rate limit per IP address (seconds)
RATE_LIMIT_PER_KEY
Rate limit per SSH key (seconds)
EXCLUDED_KEY_COMMENTS
Comma-separated list of excluded key comments
LOG_LEVEL
Logging level: DEBUG, INFO, WARNING, ERROR
